# Academia de formación
## Introducción
El objetivo de este caso práctico es analizar las matrículas de una academia de formación utilizando Power Query y Power BI.
## Enunciado
Se proporcionan tres tablas en un libro de Excel: Matriculas, Cursos y Alumnos. La tabla Matriculas contiene información sobre las matrículas, la tabla Cursos contiene información sobre los cursos y la tabla Alumnos contiene información sobre los alumnos.
## Fase 1 - Limpieza con Power Query
Se deben limpiar las tablas Matriculas y Cursos para eliminar errores de tecleo y formatos mezclados.
## Fase 2 - Relacionar y graficar en Power BI
Se deben crear relaciones entre las tablas y graficar la información para analizar las matrículas.
## Gráficos
* Matrículas por curso
* Importe total por categoría
* Reparto por ciudad del alumno
* Comparación por sexo del alumno
* Evolución de matrículas por mes
## Tarjetas (KPI)
* Importe total
* Número de matrículas
* Número de alumnos
## Segmentación (slicer)
* Filtro por Categoría o Estado
